# 🚀 모멘텀 돌파 PRO: 암호화폐 자동매매 플랫폼

**모멘텀 돌파 PRO**는 강력한 추세 추종 알고리즘과 정교한 리스크 관리 시스템을 결합한 지능형 암호화폐 자동매매 플랫폼입니다. 백테스팅을 통한 검증부터 실전 매매까지, 데이터에 기반한 투자를 지원합니다.

---

## 🔥 핵심 전략 소개: "최강의 창과 방패"

본 플랫폼은 시장 상황에 따라 선택 가능한 두 가지 특화된 **'모멘텀 돌파 PRO'** 전략을 제공합니다.

### 1. 🛡️ PRO 안정형 (Momentum Stable)
*   **목표:** 하락장 철벽 방어 및 장기 우상향
*   **특징:** 엄격한 거래량 필터(2.1배↑)와 타이트한 트레일링 스톱(1.8x ATR) 적용
*   **성과:** 
    *   **최근 6개월(하락장):** **-3.96%** (원금 보존 수준의 압도적 방어)
    *   **2020년~현재(장기):** **+70.60%** 수익 달성

### 2. ⚔️ PRO 공격형 (Momentum Aggressive)
*   **목표:** 강세장 수익 극대화 및 자산 증식
*   **특징:** 장세 강도(ADX > 35)에 따른 **2배 비중 동적 베팅(Dynamic Risk)** 및 여유로운 트레일링 스톱(2.5x ATR)
*   **성과:**
    *   **최근 6개월(하락장):** -19.09%
    *   **2020년~현재(장기):** **+81.67%** (최고 수익률 기록)

## 🛠️ 주요 기능
- **멀티 코인 포트폴리오 백테스팅:** 비트코인(BTC), 이더리움(ETH) 등 여러 코인을 동시에 시뮬레이션하여 통합 자산 관리 최적화.
- **실시간 트레일링 스톱 (Trailing Stop):** 수익 발생 시 자동으로 손절 라인을 끌어올려 수익을 보존하고 손실을 최소화.
- **다이나믹 리스크 스케일링:** 추세 강도(ADX)를 분석하여 강세장에서 자동으로 투자 비중을 확대.
- **FastAPI & Next.js 기반:** 빠르고 안정적인 백엔드와 직관적이고 세련된 대시보드 UI 제공.


## 🚀 시작하기

### 1. 백엔드 설정 (Python 3.10+)
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### 2. 프론트엔드 설정 (Node.js 18+)
```bash
cd frontend
npm install
npm run dev
```

### ⚠️ 주의사항
- 실제 매매 시에는 `backend/core/config.py`에 거래소 API 키를 설정해야 합니다.
- **API 키는 절대 깃허브 레포지토리에 노출되지 않도록 `.gitignore` 설정을 확인하세요!**
